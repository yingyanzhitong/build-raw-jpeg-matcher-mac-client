## ADDED Requirements

### Requirement: 用户可以选择当前方向的导出目录
系统 MUST允许用户在导出前选择且仅选择一个目标目录，并为两个方向分别保存当前会话中的导出目录。

#### Scenario: 选择导出目录
- **WHEN** 用户为当前方向选择一个导出目录
- **THEN** 系统保存并展示该目录

#### Scenario: 取消导出目录选择
- **WHEN** 用户打开导出目录选择器后取消操作
- **THEN** 系统保留当前方向原有的导出目录

### Requirement: 系统按当前方向导出已选候选
系统 MUST仅导出状态为 matched 或 confirmed 的唯一已选候选：“图片 → RAW”导出 RAW，“RAW → 图片”导出 JPG、JPEG 或 PNG；导出到同一目标目录时必须保留候选原文件名且不保留源目录层级。

#### Scenario: 导出对应 RAW
- **WHEN** 用户在“图片 → RAW”方向导出可用结果
- **THEN** 系统将每个可导出的唯一 RAW 候选复制到当前方向的目标目录

#### Scenario: 导出对应图片
- **WHEN** 用户在“RAW → 图片”方向导出可用结果
- **THEN** 系统将每个可导出的唯一图片候选复制到当前方向的目标目录

#### Scenario: 多个输入指向同一候选
- **WHEN** 多个匹配结果选择了同一个规范化候选路径
- **THEN** 系统最多处理和统计该物理候选一次

### Requirement: 未找到和未解决冲突不得导出
系统 MUST跳过 missing 和未确认的 conflict 结果，并在日志及摘要中分别记录跳过数量。

#### Scenario: 结果包含未找到项
- **WHEN** 用户导出包含 missing 结果的任务
- **THEN** 系统跳过这些结果并记录未找到数量

#### Scenario: 结果包含未解决冲突
- **WHEN** 用户导出包含尚未确认候选的 conflict 结果
- **THEN** 系统跳过这些结果并记录未解决冲突数量

#### Scenario: 冲突已经确认
- **WHEN** 用户已经为 conflict 结果确认一个候选
- **THEN** 系统将该 confirmed 结果纳入导出

### Requirement: 导出不得覆盖已有文件
系统 MUST在复制前检查目标路径；目标不存在时执行复制，目标存在且内容相同时不重复复制，目标存在但内容不同时不得覆盖。

#### Scenario: 目标文件不存在
- **WHEN** 候选目标路径尚不存在
- **THEN** 系统复制文件并将其计入实际复制数量

#### Scenario: 目标存在相同内容
- **WHEN** 目标文件已存在且与候选内容完全相同
- **THEN** 系统不重复复制，将其计入“已存在相同文件”而非实际复制数量

#### Scenario: 目标存在不同内容
- **WHEN** 目标文件名已存在但内容与候选不同
- **THEN** 系统不覆盖目标，并将其计入文件名冲突

### Requirement: 单个源文件错误不得终止整批导出
系统 MUST逐项处理导出候选；候选在匹配后被删除、失去读取权限或复制失败时，系统必须记录该项失败并继续处理其他候选。

#### Scenario: 一个候选在导出前被删除
- **WHEN** 批量导出中的一个候选路径已经不存在
- **THEN** 系统记录该候选失败并继续导出其余候选

#### Scenario: 一个候选无法读取
- **WHEN** 系统无法读取某个候选
- **THEN** 系统记录错误原因并继续处理其余候选

### Requirement: 导出完成后展示方向化摘要
系统 MUST在导出完成后展示实际复制、已存在相同文件、跳过未找到、跳过冲突、文件名冲突和源文件失败的数量，并使用当前方向对应的文件类型文案。

#### Scenario: RAW 导出完成
- **WHEN** “图片 → RAW”方向导出结束
- **THEN** 系统展示 RAW 文件导出摘要和目标目录

#### Scenario: 图片导出完成
- **WHEN** “RAW → 图片”方向导出结束
- **THEN** 系统展示图片文件导出摘要和目标目录
